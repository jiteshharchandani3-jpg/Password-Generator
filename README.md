## ⚙️ How It Works

1. Select your desired password length.
2. Choose which character types to include.
3. Optionally exclude similar-looking characters.
4. Select a generation mode:
   - Random Password
   - Pronounceable Password
   - PIN
5. Click **Generate Password**.
6. View password strength and entropy.
7. Copy the password with one click.
8. Recently generated passwords are stored locally for quick access.

---

## 🔒 Security

This project uses the **Web Crypto API (`crypto.getRandomValues()`)** instead of `Math.random()` to generate cryptographically secure random passwords.

Passwords are generated entirely in your browser, and no data is sent to any server.
